Where to find additional documentation?

Additional documentation is available in the form of online technical papers. It can be accessed in one of two ways.
  • You can reach it on your own computer in the ./doc/additional_doc directory . However, it is not automatically included in the documentation for the full OrbFit distribution; if you have not downloaded it yet, you should get the archive from the documentation directory: get the file additional_doc.tar.gz, and unpack it in the root OrbFit directory.
     
  • If you do not wish to download the additional documentation immediately, you may browse the information on the World Wide Web by pointing your browser to   http://newton.dm.unipi.it/orbfit/OrbFit/doc/additional_doc/.
